# File server
#### Local file streaming that works!

## Install
```bash
$ npm i -g filess
```

## Usage
```bash
$ filess
```

And for more help
```bash
$ filess -h
```

```bash
A file server/streaming that works.

  Usage
    $ filess <options>

  Options
    --dir, -d   Set the directory you wanna serve, default: ~/
    --port, -p  Set the port, default: 3030
    --help, -h  Showing this help menu
  
  Examples
    $ filess
    $ filess -d ~/Videos
    $ filess -p 3000
    $ filess -d ~/ -p 3000
```
